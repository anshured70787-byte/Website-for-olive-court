# The Olive Court Website

Production-ready Next.js website for **The Olive Court, Jodhpur** with a dark luxury editorial food-commerce aesthetic.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Pages

- `/` — premium restaurant landing page with hero, about, features, signature dishes, cuisine, gallery, reviews, reservation CTA and contact sections.
- `/menu` — categorized menu page with sticky filters and elegant menu cards.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Assets

The `public/images` directory contains lightweight SVG placeholder art for food, ambience, drinks, rooftop and live-music imagery. Replace these files with real restaurant photography when available while keeping the same filenames to avoid code changes.
